# Replacing NA values; number column(s) with ave. value while a the category column(s) with a dummy text, assigning it to 'df_cleanData'
# We will calculate the mean values inside a dictionary the .describe() function will help us see if we have the right value

# We will use a dictionary to change multiple NA values
df_cleanData = df_data.fillna({"Age" : round(df_data['Age'].fillna(value = df_data['Age'].mean())),
                               "Salary(k$)" : round(df_data['Salary(k$)'].fillna(value = df_data['Salary(k$)'].mean())),
                               "Purchased" : 'lorem'})

To plot a balanced graph we need to re-order the horizontal labels which are categorical data by using the function scale_x_discrete()

The limits argument determines the ordered labels in the list.

# Plot bar graph showing Country against Total Salary(k$) using

# ploting the graph
(
    ggplot(df_plot1)
    + aes(x = 'Country', y = 'Salary(k$)')
    + labs (
        x = 'Country Represented',
        y = 'Total Salary(k$)',
        title = 'Country against Total Salary(k$)'
        )
    + theme(axis_text_x = element_text(angle = 45))
    + geom_bar(fill = 'blue', stat = 'identity')
    + scale_x_discrete(limits = df_plot1['Country'].tolist())
)
